# Proyecto 1: Protección Medioambiental 🌍
Este repositorio contiene el trabajo colaborativo para el **Tema 4** de la asignatura. El objetivo es analizar el impacto de la tecnología en el medio ambiente y proponer soluciones sostenibles.

## 🚀 Ver la página web
Puedes ver el resultado final del proyecto aquí:
👉 **[https://dmarsue242.github.io/Informatica-Medioambiental/]**

## 📚 Contenidos del Proyecto
La página aborda los siguientes puntos clave:
1. **¿Qué es la contaminación ambiental?**: Introducción y conceptos básicos.
2. **Residuos informáticos**: El problema del e-waste.
3. **Obsolescencia programada**: El ciclo de vida de los productos.
4. **Informática ecológica**: Hacia un futuro sostenible.
